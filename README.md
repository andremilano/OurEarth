# 🌿 EarthGuard
Interactive Environmental Awareness Website

## 📖 Project Overview
EarthGuard adalah website awareness bertema lingkungan dengan pendekatan natural earthy aesthetic yang bertujuan meningkatkan kesadaran masyarakat terhadap isu lingkungan melalui storytelling visual yang hangat, modern, dan inspiratif.

Website ini dirancang sebagai pengalaman scroll yang mengalir, bersih, dan emosional, dengan fokus pada visual impact dan user experience.

---

## 🌍 Background
Permasalahan lingkungan seperti pencemaran plastik, deforestasi, dan perubahan iklim semakin meningkat setiap tahun. Namun, kesadaran masyarakat masih rendah karena kurangnya media edukasi yang menarik dan interaktif.

EarthGuard hadir sebagai media digital yang menggabungkan desain modern, animasi halus, dan storytelling untuk menyampaikan pesan lingkungan secara lebih engaging.

---

## ✨ Features & Interactions

- Sticky navbar dengan efek perubahan saat scroll
- Hero section full-screen dengan animated text reveal
- Count-up animation pada statistik lingkungan
- Scroll reveal animation berbasis Intersection Observer
- Interactive cards dengan hover lift effect
- Timeline edukatif dengan animasi progres saat scroll
- Smooth scrolling experience
- Fully responsive (Desktop, Tablet, Mobile)

---

## 🛠 Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- Intersection Observer API
- Custom animation (CSS & JS)

No backend. Static website only.
